Feynman Loop

你先问，我讲清；我再问，你讲回。

这个训练器面向刚接触 vibe coding 的人：每次围绕一个问题，用“简单解释、类比、约束、例子、反问评估”的节奏，把模糊经验变成能复述的方法。

第 1 步

先问一个你想弄懂的问题

你可以直接输入自然语言。系统会匹配最接近的闪卡主题，给出新手版解释，再进入反问训练。

目标不是背定义，而是能把一个问题讲成：是什么、为什么、怎么做、怎么检查。
Loop 是核心节奏

Vibe coding 不是一次问对，而是一轮一轮逼近。

① 意图 → 生成

写清楚目标和约束，让 AI 生成第一版。不要追求一次完美，先跑起来再说。

② 运行 → 观察

每次改动后立刻跑起来看结果。不跑就反馈，等于蒙眼开车——你不知道 AI 改对了没有。

③ 反馈 → 再生成

说出观察到的问题 + 希望的变化 + 保留什么。一次只改一件事，这样你知道哪里有效。

④ 保存 checkpoint

每次”现在是好的”就存一版。Git commit 或复制文件都行。下一步改错了，你还能退回来。

Loop 卡住了怎么办

改了 3 次还没进展？停下来重新描述：我现在有什么、我想得到什么、它们之间差了什么。

工具是配角，loop 是主角

Codex、VS Code、PowerShell 各有分工，但它们都服务于同一件事：让 loop 转得更快、更准。
